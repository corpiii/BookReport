# BookReport 소개
꾸준한 독서 습관을 기르기 위해 알림을 설정하고 하루에 읽은만큼만 독후감을 작성합니다.

## 프로젝트 영상
- 구글 로그인
- 책 생성
- 독후감 
  - 독후감 생성
  - 독후감 수정
  - 독후감 삭제
- 알림 설정
- 알림 해제
- 로그아웃

위의 순서로 진행됩니다.
